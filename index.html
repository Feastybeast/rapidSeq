General Formatting

  • Object names may not contain '?'s.
  • Object 1 => Object 2: message passed for fat lines.
  • Object 1 <- Object 2: message passed for thin lines.

Lifecycle Hints

  • Unless specified otherwise, objects exist from instantiation until termination.
  • "Controller" .. "/Controller" to indicate the duration of some object.
  • "Database/" to indicate an immediate termination of an object
